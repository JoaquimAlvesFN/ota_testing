
# Online Travel Agency

Base Project for an online travel agency


[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)


## Deployment

To deploy this project run

```bash
  docker-compose up
```


## API Reference

#### Get all Users

```http
  GET /api/users
```

#### Get all Travels

```http
  GET /api/travels
```


## Features

- CRUD Users
- CRUD Travels
